# SL96: 1S 2-Inch 1002 Toothpick FPV Quad

An ultra-light 1S 2-inch toothpick build focused on very high thrust-to-weight, fast prop response, and minimal all-up weight.

## Build Concept

This quad is designed around a lightweight carbon frame, high-KV 1002 motors, and 2-inch bi-blade props.  
The goal is a very responsive 1S build with aggressive acceleration, low prop spool delay, and a high-pitched “angry micro” sound.

Estimated all-up weight with a 1S 580 mAh LiHV battery is around **39–40 g**.
